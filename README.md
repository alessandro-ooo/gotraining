# Welcome to gotraining!

Gotraining is a simple application that allows Personal Trainers to build workout plans for their customers without the hassle of learning complicated applications to generate PDFs. This applications allows new Personal Trainers to create PDFs without being charged and it's ADS FREE!

## Current features
- Workout plan manager: create, edit and delete plans in just a few click.
<p align="center">
  <img src="https://i.imgur.com/zIFAqop.png" alt="logo"/>
</p>


- PDF editor: edit the graphic of the tables, add your own logo at the top of the PDF with a basic editor.
